```css {7-13} /display: inline/
.gradient-text {
  --gradient-direction: to right;
  --gradient-from: #5237f9;
  --gradient-to: #00b8b9;
  --gradient-stops: var(--gradient-from), var(--gradient-to);
  display: inline;
  background-image: linear-gradient(
    var(--gradient-direction),
    var(--gradient-stops)
  );
  -webkit-text-fill-color: transparent;
  -webkit-background-clip: text;
  background-clip: text;
}
```

Transform text into a stunning gradient with this CSS class. Simply define the direction, start color, end color, and stops of the gradient as custom properties, and the text will seamlessly blend into the gradient background. Supports all major browsers, including webkit-based browsers, with transparent text fill and background-clip properties set.

## Browser Compatibility

- [`background-clip:text`](https://caniuse.com/?search=Background-clip%3A%20text)
- [`-webkit-text-fill-color`](https://caniuse.com/?search=CSS%20property%3A%20-webkit-text-fill-color)
