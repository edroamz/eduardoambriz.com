---
title: WithRequired Type
description: Easily create new types with mandatory specified properties
icon: typescript
---

```ts
type WithRequired<T, K extends keyof T> = T & { [P in K]-?: T[P] };
```

Type utility `WithRequired` enforces mandatory properties in type `T` by creating a new type with required properties from `K`. The result is a combination of `T` and a new object with required properties.
